## 使用教程

1. 注册瑞思账户（已注册跳过）[打开链接](https://user.3r60.top/register/)
2. 打开瑞思开发者中心，并复制您的Token [打开链接](https://user.3r60.top/dev/)
3. 依次点击 应用列表->新建应用->输入您的应用信息->创建应用->进入应用信息页面并复制 Appid
4. 下载仓库中的index.php文件，放入您的网站中，推荐使用PHP7
5. 编辑index.php文件，在首部APPID、TOKEN、ENCYC变量处填写您的开发凭据（appid+token）与加密码（加密码用于id对应随机码的文件名）
6. 如何访问我的文件： 您的网站目录/user/随机ID/config.json
